# Customer Churn Prediction Dashboard

## 📌 Project Overview

Customer churn is one of the major challenges faced by subscription-based businesses such as telecommunications, banking, insurance, and SaaS companies. This project aims to predict whether a customer is likely to leave a company based on customer demographics, service usage, contract details, and payment information.

A Machine Learning model was developed and deployed using Streamlit to provide real-time churn predictions through an interactive dashboard.

---

## 🎯 Business Objective

The objective of this project is to identify customers who are likely to churn so that businesses can take proactive measures to improve customer retention, reduce revenue loss, and enhance customer satisfaction.

---

## 📂 Dataset

**Dataset:** Telco Customer Churn Dataset


## 🛠️ Project Workflow

### 1. Data Collection
- Imported the Telco Customer Churn dataset.
- Loaded the dataset using Pandas.

### 2. Data Cleaning
- Checked for missing values.
- Removed duplicates.
- Verified data types.

### 3. Exploratory Data Analysis (EDA)
Performed data visualization using Matplotlib and Seaborn to understand customer behavior and identify factors influencing churn.

Key analyses:
- Churn distribution
- Contract Type vs Churn
- Internet Service vs Churn
- Payment Method vs Churn
- Tech Support vs Churn
- Tenure Analysis

### 4. Data Preprocessing
- One-Hot Encoding of categorical features.
- Feature Scaling using StandardScaler.
- Train-Test Split.

### 5. Model Building
The following machine learning models were trained and evaluated:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- KNeighborsClassifier

### 6. Model Selection
Logistic Regression achieved the best performance and was selected as the final model.

### 7. Model Deployment
Built an interactive Streamlit dashboard to allow users to input customer details and receive churn predictions in real time.
